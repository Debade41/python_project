Telegram-бот, который анализирует текстовые сообщения на наличие упоминаний валют (например, "100к долларов" или "2млн евро") и автоматически конвертирует их в другие валюты, используя актуальные курсы ЦБ РФ. Поддерживаются суффиксы вроде "к" (тысячи), "м" (миллионы) и т д. Система включает backend API для обработки, базу данных PostgreSQL для хранения истории конвертаций и веб-дашборд на Streamlit для просмотра распознанных валют и истории.
Основные функции
Парсинг и конвертация текста без команды,
конвертация определенной валюты в другую по команде /convert <сумма> <из> <в>,
просмотр истории конвертаций /history <число записей>
Запуск docker-compose up --build
